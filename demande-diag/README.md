# Demande de diag (Web Components)

## Création d'une nouvelle version

Pour créer une nouvelle version, il faut dans un premier temps modifier la version dans le fichier
`package.json` de ce répertoire puis exécuter manuellement le workflow (`Déploiement WebComponent demande de diag`)
dans GitHub.

Celui-ci va copier les fichiers des répertoires `dist-demo` et `dist-prod` sur le cellar de MSC.
