<p>Les documents présents dans cette page présentent les bonnes pratiques relatives au déploiement sécurisé des pare-feux Stormshield Network Security (SNS), en version physique ou en version virtuelle. Les recommandations détaillées dans ce document traitent des fonctions :</p><ul><li>d’administration ;</li><li>de filtrage ;</li><li>de chiffrement IPsec ;</li><li>de supervision ;</li><li>de sauvegarde ;</li><li>de journalisation.</li></ul><p>Ces documents viennent en complément d'autres publications de l’ANSSI relatives aux pare-feu, en particulier :</p><ul><li>la note technique «<a href="/guides/definition-dune-politique-de-pare-feu">Recommandations pour la définition d'une politique de filtrage réseau d'un pare-feu</a>» ;</li><li>le guide «<a href="/guides/recommandations-relatives-linterconnexion-dun-si-internet">Recommandations relatives à l'interconnexion d'un SI à Internet</a>».</li></ul><p>Les recommandations présentes dans la version 3.0 du guide sont applicables à la version 3.7.17 de Stormshield Nework Security (SNS).</p>
